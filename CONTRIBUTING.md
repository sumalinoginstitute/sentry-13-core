# Contributing to SENTRY-13

Thank you for your interest in contributing to the Sumalinog Institute open-core repository.

## Developer Certificate of Origin (DCO)
To maintain the legal integrity of this public good infrastructure under the Apache License 2.0, all pull requests must include a "Signed-off-by" line in the commit message, certifying that you have the legal right to submit the code.

Example commit:
`Git commit -s -m "Optimized multiplicative loop execution latency"`
